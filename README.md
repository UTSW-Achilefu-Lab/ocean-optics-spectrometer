# Ocean Optics spectrometer live viewer
> Ian Zurutuza - June 2025

Simple data viewer for ocean optics spectrometers

## Usage

**Windows (powershell)** (untested)
```
# confirm you have python
python3 --version 

# create virtual environment named ".venv"
python3 -m venv .venv
./.venv/bin/Activate.ps1

# install dependencies seabreeze and matplotlib
pip install seabreeze matplotlib

# run program
python test.py
```
